1. PYTHON server

Then navigate to the directory ./python.
Run the following commands:

pip install flask
pip install flask-cors
pip install python-dotenv
pip install requests
python classification.py

You should then see the server is running in the console.


3. MAIN WEBSITE

Start a new powershell and navigate to the directory ./stoado-react-app
Run:

npm install
npm start

That should take you to the main website.
